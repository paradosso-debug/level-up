# ZenFlow Studio — NO JS (Step-by-step Comment Style)

This version matches your **“Young Miko” instructional structure**:
- HTML steps are written as **big numbered comment blocks**
- CSS steps are written as **rule-by-rule comment blocks**
- **No JavaScript** (no `<script>` tags)

## What mentees practice
- Semantic layout: `header`, `nav`, `main`, `section`, `footer`
- Sticky header navigation with jump links (`href="#sectionId"`)
- Responsive layout using CSS Grid
- Card grids + gallery tiles
- Clean UI tokens (colors, spacing) using CSS variables

## Run
Open:
- `mentor/index.html` for the finished version
- `mentee/index.html` for the instruction-only version

Assets are in `/assets` (header image + logo images).
