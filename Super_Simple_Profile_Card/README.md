# Super Simple Profile Card (HTML + CSS)

## What you’re building
A small **profile card** with:
- A circle avatar
- A name + role
- A short bio
- Two buttons (Message / Follow)

This project is **super beginner-friendly** and focuses on clean structure + basic styling.

---

## What you’ll learn (and practice)
### HTML concepts
- `<!DOCTYPE html>` and the basic page structure (`<html>`, `<head>`, `<body>`)
- Using a main wrapper (`<main>`) to hold your content
- Building a “card” with a `<section>`
- Adding text with headings and paragraphs (`<h1>`, `<p>`)
- Adding an image with `<img>`
- Grouping items with `<div>`
- Creating buttons with `<button>`

### CSS concepts
- Setting a global font on the page
- Centering content using Flexbox
- Styling a “card” (background, border, rounded corners, shadow)
- Making the avatar image circular
- Spacing with padding, margin, and gap
- Button styling + hover effect

---

## How to run it
1. Open the project folder.
2. Double-click `index.html` to open it in your browser.
